Every OneInbox API request uses one credential: your **API key** (`oi_live_...`).

---

## Get your API key

1. Log in to the **OneInbox dashboard** (URL provided during onboarding)
2. Open **API Keys**
3. Click **Create API key**, give it a name, and copy the key

<Warning>
The full API key is shown **only once**. Store it somewhere safe (password manager, secrets vault). If you lose it, create a new key and revoke the old one from the dashboard.
</Warning>

Accounts are created by the OneInbox team during onboarding — there is no self-service sign-up in the API docs.

---

## How to send it

Add this header to every request:

```
Authorization: Bearer <api_key>
```

Example:

```bash
curl http://13.207.23.185:8000/v1/agents \
  -H "Authorization: Bearer <api_key>"
```

---

## Common mistakes

### Missing or wrong API key

**Symptom:** `401 Unauthorized` on any endpoint.

**Fix:** Check you copied the full key (starts with `oi_live_`) and used `Authorization: Bearer <api_key>`.

### Lost your API key

You cannot retrieve it again. Log in to the dashboard → **API Keys** → create a new key and revoke the old one if needed.
