OneInbox は **音声 AI エージェント**用の REST API です。エージェントを作成し、電話またはブラウザでテストできます。

**初めての方は** [クイックスタート](/jp/quickstart/first-agent) から — 各 curl をコピーし、ID を保存して次のステップへ。

---

## 5つのステップ（順番通り）

| ステップ | 内容 | 保存する ID |
| --- | --- | --- |
| 1 | サインアップ + API キー | `api_key` |
| 2 | OpenAI キーを保存 | `credential_id` |
| 3 | LLM モデル（プロンプト） | `llm_id` |
| 4 | エージェント（STT + LLM + TTS） | `agent_id` |
| 5 | ブラウザテスト通話 | meet リンクを開く |

<Info>
ドキュメントでは **インテグレーション** と表記。API パスは `/v1/credentials` です。
</Info>
