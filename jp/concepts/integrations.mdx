## インテグレーションとは

**インテグレーション**は、OneInbox 内に安全に保存されるサードパーティ API キーです。

- OneInbox がキーを **暗号化** して保存
- 作成後、生のキーは **返却されない**
- 他の API 呼び出しでは `id` で参照

---

## いつ必要か

クイックスタートには **不要** — OneInbox がデフォルトの LLM / STT / TTS を提供。**自社のベンダー課金** やカスタムボイスが必要なときに追加します。

| 用途 | タイミング |
| --- | --- |
| **自社 LLM 課金** | 自社 LLM アカウント（BYOK）を使う場合 |
| **テレフォニー** | 電話通話（発信・着信） |
| **カスタムボイス** | カスタムまたはクローンボイスのインポート |

---

## 作成方法

```bash
curl -X POST https://api.oneinbox.ai/v1/credentials \
  -H "Authorization: Bearer <api_key>" \
  -H "Content-Type: application/json" \
  -d '{
    "name": "OpenAI Production",
    "provider": "openai",
    "api_key": "<your_provider_api_key>"
  }'
```

---

## インテグレーションのつながり

```
インテグレーション（任意 BYOK）
        │
        ▼
一致する "provider" の LLM モデル
        │
        ▼
エージェント（llm_id）→ 通話
```

テレフォニーインテグレーションは、番号検索・登録の `credential_id` 経由で [電話通話](/jp/guides/phone-calls) を支えます。

---

## インテグレーションの管理

```bash
curl https://api.oneinbox.ai/v1/credentials \
  -H "Authorization: Bearer <api_key>"
```
