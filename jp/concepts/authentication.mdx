すべての OneInbox API リクエストには 1 つの認証情報 — **API キー** — が必要です。

---

## API キーの取得

1. [OneInbox ダッシュボード](https://oneinbox-dashboard.vercel.app) にログイン
2. **API Keys** を開く
3. **Create API key** をクリックし、名前を付けてキーをコピー

ダッシュボードでいつでもキーの作成・失効ができます。

---

## 送信方法

すべてのリクエストにこのヘッダーを追加:

```
Authorization: Bearer <api_key>
```

例:

```bash
curl http://13.207.23.185:8000/v1/agents \
  -H "Authorization: Bearer <api_key>"
```

---

## よくあるエラー

### API キー欠落または誤り

**症状:** 任意のエンドポイントで `401 Unauthorized`。

**対処:** ダッシュボードから完全なキーをコピーし、`Authorization: Bearer <api_key>` を使用。

### 失効または紛失したキー

ダッシュボードで新しいキーを作成し、必要に応じて古いキーを失効してください。
