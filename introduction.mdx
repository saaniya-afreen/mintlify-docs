## What is OneInbox?

OneInbox is a **voice orchestration platform** for building AI agents that handle live voice conversations over the phone and the web.

This documentation covers the **REST API** and how to integrate OneInbox into your product. You configure agents, models, and calls with API requests — OneInbox handles real-time audio, telephony, and scaling.

---

## How voice agents work

Every call runs a loop until the conversation ends:

```mermaid
flowchart LR
  User([Caller speaks]) --> STT[Speech-to-text]
  STT --> LLM[Language model]
  LLM --> TTS[Text-to-speech]
  TTS --> User
```

| Layer | You configure | Default |
| --- | --- | --- |
| **STT** | Provider, model, language | OneInbox default |
| **LLM** | Model, system prompt, tools, knowledge bases | OneInbox default |
| **TTS** | Provider, voice, speed | OneInbox default |

OneInbox connects these layers in real time. You focus on **what the agent says and does**.

---

## What you can build

- **Outbound callers** — your agent dials leads, qualifies them, or sends reminders
- **Inbound support** — assign a phone number; callers reach your agent 24/7
- **Web voice** — embed a voice widget on your site with the OneInbox Web SDK
